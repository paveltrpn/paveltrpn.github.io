Некоторые комманды Linux

Выключить компьютер: 

                $ sudo shundown -h now

Лист всех файлов в каталоге (включая скрытые) с размерами файлов в человекочитаемом формате: 

                $ ls -a -l *

Поиск пакета в репозиториях: 

                $ sudo apt-cache search glut

Проверка версии пакета в репозиториях. Можно сделать с помощью apt-cache или apt-get. 

                $ sudo apt-cache show "name" | grep -i version 
                или
                $ sudo apt-cache policy "name"
                с помощью ключа -s - симуляция установки
                $ sudo apt-get -s install mc
